README for inxi branch two development branch.

=====================================================================
This is a development branch, and corresponds to the inxi update 
option: -! 12

The direct download path for this branch is:

https://github.com/smxi/inxi/raw/two/inxi

DO NOT PUSH ANYTHING TO MASTER FROM THIS BRANCH!!!!

DO NOT MERGE THIS BRANCH WITH ANYTHING ELSE!!! 

This branch is usually out of date, and is only relevant when active
development is going on.

The main purpose for this branch is to have the ability to test 
changes on various systems by use of the inxi -! 12 command.

You can verify you are running the branch two version because it will
show -b2 after the main inxi version number, like so:

2.3.40-b2

=====================================================================
